ABOUT SoDoHo

SoDoHo is our project.

SoDoHo was a server we made on April since we wanted a place to play and I (jnsl) knew I could do the server back end, ketch could do the Discord server, be gone with the third party apps, and olozzy as the community leader. CGSDH was a group we were all already part of so we named it after that. SoDoHo had gone thru multiple months on a host we were just testing but we decided to move to this new host. I also took this opportunity to add more integrations and features to make SoDoHo what it is today.

CGSDH Server





SoDoHo Started from a fun project to something that blew up. We did not expect this. But this extended growth pushed us to do more and now we are here. While our 1st Winter Tournament did not go as well as we wanted, it was a good learning lesson for us all. It also pushed me to start this new host and these new integrations as part of the project. We aim to make the experience better for the end user.
